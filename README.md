# Translation App with Flask

A language translation web app with text-to-speech.

## Setup
1. Install dependencies:
         pip install flask googletrans==4.0.0-rc1 gtts

2.Run:

 In spyder: Run application.py

3. if the output is like this:
    %runfile C:/Users/pavit/Downloads/translator_project/application.py --wdir

 * Debug mode: on
WARNING: This is a development server. Do not use it in a production deployment. Use a production WSGI server instead.
 * Running on all addresses (0.0.0.0)
 * Running on http://127.0.0.1:5000
 * Running on http://192.168.249.183:5000
Press CTRL+C to quit

Flask is running

4. Use the http://localhost:5000 link or the output console link for the language translator application.
